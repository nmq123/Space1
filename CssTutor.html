Css instructions

  1. Css basics

    • Inline: To add properties to inline text, use structure:
      <h1 style="color:blue">Css instructions</h1>
    • Internal: To add properties to everything for a specific type of entity, for example, adding <style> to the <head> html tag:
      <style>
    h2,p{
        color:red;
        font-style:italic;
    }
</style>
      will add color, font style to the <h2> and <p> tag
    • External: To add properties to all html files in a project, create a new style.css file and add properties to it, eg:
      h2,p{
    color:red;
    font-style:italic;
}
      Then, in the main html file, add to the <head> html tag the link: <link rel="stylesheet" href="style.css">
      or simply enter style:css to fill the code
    • When all 3 of this are present, the priority is: Inline >> Internal >> External

  2. Some common properties

    1. color: Change the color of object, example:
      2. h1 {color:red;}
    2. font-style: Change the style of text, example:
      3. h1 {font-style:Italic;}
    3. font-family: Change the font of text, example:
      4. h1 {font-family:'Times New Roman';}
    4. text-transform: Capitalize or un-capitalize text, example:
      5. h1 {text-transform:capitalize;}
    5. text-align:Align object, example:
      6. h1 {text-align:left;}
    6. font-size:Change font size, example:
      7. h1 {font-size:28px;}
    7. opacity: Change the transparency of object, example:
      8. h1 {opacity: 60%;}
    8. visibility:Hide or show object (hide also block interaction), example:
      9. h1 {visibility: visible;}
    9. display: Display object or not, example:
      10. h1 {display:none;}
      • Distinguishing Opacity 0, display none and visibility hidden
    10. background-color:Add background color for text, example:
      11. h1 {background-color: blue;}
    11. background-image: Add background image, example:
      12. body {background-image:url(/img/Earth.jpg);}
    12. transition: Time taken to transit upon display update for an object (for animation, and does support opacity and visibility), example:
      13. img {
  transition: 1.5s;
}

input:checked ~ img {
  opacity: 0;
  visibility: hidden;
}
    13. favicon: Add icon for website, place the link to <head> of html:
      14. <link rel="shortcut icon" href="https://github.com/fluidicon.png" type="image/x-icon">
      to add ico file for web icon
    14. user-select:Allow or disallow user to select text
      15. user-select: none

  3. Combinators

    1. ID selector:This will mark a specific part with an ID (IDs must be unique) and call them out in css file, example:
      2. In html file:
      <h2 id="Anyname">Test text</h1>
      In css file:
      #Anyname {color="red";}
      We can also type in html file, for example "h2#Id1" and it will auto creat h2 with id
    2. Class selectorThis can select multiple parts with same class and call them out in CSS file, example:
      3. In html file:
      <h2 class="Name">Test text1</h2>
<h2 class="Name">Test text2</h2>
      In css file:
      .Name {color="red";}
    3. Combinator selectors:
      • Use * to add properties to everything
        • * {color: red;}
      • We can also name multiple classes to an object (separated with space) to call it out more specifically in css file, example:

        • In html file:
        <h2 class="Class1">Object1</h2>
<h2 class="Class1 Class2">Object2</h2>
        In css file:
        .Class1.Class2 {color="red";}
        This will change only Object2 leaving Object1 intact since it only have Class1
        A faster writing in html file, for example "h2.class1.class2" will auto fill the rest
      • In css file, use space to select all sub objects of a specific type inside the main object
      • Use ">" to select first level sub objects
      • Use "+" to select one specific object if placed right next to it
      • Use "~" to select all specific object if placed after it (regardless of discontinuity)
        • In html file:
        <p>
    <h2>Header1.1</h2>
    <h2 class="C1">header1.2</h2>
    <h2 class="C1 C2">header1.3</h2>
    <h2>Extra header1</h2>
    <h3>Header2.1</h3>
    <h3>header2.2</h3>
    Main paragraph<br>
    line1<br>
    line2<br>
    <div>
        <p>Paragraph1.1</p>
        <p>Paragraph1.2
            <blockquote>Paragraph2.1
                <p>paragraph3.1(p inside blockquote)</p>
            </blockquote>
        </p>
    </div>
</p>

        In css file:
        div p{color:blueviolet;} will paint all <p> object inside <div>, including paragraphs(1.1, 1.2, 3.1), leaving paragraph2.1 intact (since its inside <blockquotee>)
        While:
        div>p{color:blueviolet;} will only paint paragraph1.1 and 1.2 And:
        .C2+h2{color:blueviolet;} will paint purple "Extra header1"
        while
        h2~h3{color:blueviolet;} will paint all header2 (<h3>)
    4. Selector element.class: Add .class after an object in css file to add properties to objects with same class but different tag type, example:

      5. In html file:
      <h2 class="Class1">Object1</h2>
<p class="Class1">Object2</p>
      In css file:
      p.Class1 {color="red";}
      This will change only Object2 to red, leaving Object1 intact despite same class
