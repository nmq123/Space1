  1. Tags

    2. Tag Usage Example
    <h> Highlight headers including 6 levels <h3>
    <p> Insert a paragraph <p>lorem10*3</p> will generate 3 line 10 words each
    <br> Go to the next line Put <br> at the start or end of each line
    <hr> Place a horizontal line to separate parts Put <hr> between 2 sections or at the end
    <ol> Initialize an ordered list <ol type="I">Ordered list with indexes of I,II,...</ol>
    <ul> Initialize an unordered list Type: "ul*3{item $}" to create 5 unordered lines
    <li> Insert a list item for ol and ul <ol><li>An item in the ordered list</li></ol>
    <a> Embed a link

 We can also add width or height for custom size (declare just one will auto scale the other). Here, source can also be
local using relative path like "../" (if before) or "./" (if after) or absolute path with
just "/"
    <iframe> Insert a frame of anything (video, pop-ups, images, ect) <iframe src="/img/Pendulum_Acceleration.mp4" frameborder="0">Pendulum_Acceleration/iframe>
    <pre> Insert exact text without trimming Like that of <p> tag
    <code> Insert a text to be considered as a code

    <table> Initialize a table Initialize a table then add rows <rt> or headers <th> or table data <td>
    <tr> Insert a row for the table Using rowspan and colspan can help extend the text box: <td colspan="2">This box span 2 columns</td>
    <th> Insert headers for the table (columns) Using rowspan and colspan can help extend the text box: <td rowspan="2" >This box span 2 rows</td>
    <td> Insert a data box to the table (at current row) Using rowspan and colspan can help extend the text box: <td rowspan="2" colspan="2">This box span 2 by 2</td>
